# Gustese — сайт пекарні

**ВНТУ | 4 курс | 2 семестр | Програмування Internet**

Мультисторінковий веб-сайт пекарні **Gustese** з системою авторизації та інтерактивною таблицею контактів. Проект розроблено в рамках 5 лабораторних робіт.

---

## Опис проєкту

Сайт пекарні «Pastry with love» — місце, де кожна випічка створюється з любов'ю. Пропонує свіжі інгредієнти щодня, традиційні рецепти та зручний зворотний зв'язок через контактну сторінку.

### Технології

- **HTML5** — семантична розмітка
- **CSS3** — змінні, Flexbox, адаптивність, media queries
- **JavaScript** — нативний JS + jQuery для DataTables
- **DataTables** — пошук, сортування, пагінація
- **localStorage** — збереження користувачів та контактів

---

## Лабораторна робота №1

### Мета
Ознайомитися з підходами та технологіями, що використовуються для розмітки сучасних веб-сторінок, та розмітити веб-сторінку за допомогою мови розмітки HTML5 та каскадних таблиць стилів CSS3.

### Реалізація

- **Структура сторінок:**
  - `index.html` — головна сторінка (hero-секція «Pastry with love», Art of cakes, Tasty pancakes, Breakfast, Jibarito sandwich)
  - `welcome.html` — про пекарню
  - `menu.html` — меню (сніданки, випічка)
  - `reservations.html` — резервації, години роботи
  - `news.html` — новини та події
  - `contact.html` — контактна інформація

- **Загальний дизайн:**
  - Шрифти: Playfair Display, Alegreya (Google Fonts)
  - Кольорова схема: бежевий фон (#faf7f2), акцент (#8b6914)
  - Header: логотип, навігація, соціальні іконки
  - Footer: навігаційні посилання
  - Контентні картки з зображеннями

- **Файли:** `style.css`, `script.js` (базовий функціонал меню)

## Лабораторна робота №2

### Мета
Освоїти основні методи використання каскадних таблиць стилів CSS. Навчитися вказувати властивості для елементів гіпертекстової розмітки HTML-документа, створювати свої класи стилів.

### Реалізація

- **Властивості елементів та власні класи:**
  - Класи `.layout-fixed`, `.header-inner`, `.nav-list`, `.content-card` тощо
  - Властивості кольору (`color`, `background-color`), шрифту, відступів, сіток
  - Змінні CSS (`:root`) для кольорів, шрифтів, радіусів
  - Breakpoints для планшетів та мобільних (< 992px)
  - Гнучкі сітки (grid, flexbox)

## Лабораторна робота №3

### Мета
Дослідити властивості шрифту CSS, налаштування посилань. Створення меню на основі списку.

### Реалізація

- **Властивості шрифту:**
  - `font-family` (Playfair Display, Alegreya), `font-size`, `font-weight`, `letter-spacing`
  - Підключення Google Fonts

- **Налаштування посилань:**
  - Стилі `a` та `a:hover`, кольори для `.nav-list a`, `.social-link`
  - Посилання в header, footer, кнопки-лінки

- **Меню на основі списку:**
  - `<ul class="nav-list">` з `<li>` та посиланнями
  - Стилізація списку (flexbox, `list-style: none`)
  - Мобільне dropdown-меню

## Лабораторна робота №4

### Мета
Оволодіти навичками керування інформаційним вмістом Web-документа засобами мови JavaScript.

### Реалізація

- **Керування DOM:**
  - `document.getElementById`, `querySelector`, `addEventListener`
  - Динамічна зміна контенту (показати/приховати блоки, оновити текст)

- **Модальне вікно:**
  - Відкриття/закриття за класом `is-open`
  - Перемикання вкладок (Увійти / Реєстрація)
  - Обробка форм через `submit`

- **Керування таблицею контактів:**
  - CRUD (додати, редагувати, видалити рядки)
  - Валідація введених даних (Regex для телефону та email)

---

## Лабораторна робота №5

### Мета
Отримати навички створення веб-застосунків за сценарної мови JavaScript, бібліотеки jQuery та технології Ajax.

### Реалізація

- **jQuery:**
  - `$('#contactTable').DataTable()` — ініціалізація DataTables
  - `$(document).on('click', ...)` — делеговані події
  - `.closest()`, `.find()`, `.html()`, `.data()` — робота з DOM

- **DataTables (jQuery-плагін):**
  - Пошук, сортування, пагінація таблиці контактів
  - Українська локалізація
  - API для додавання/редагування/видалення рядків

- **Робота з даними (localStorage):**
  - Збереження користувачів та контактів без перезавантаження сторінки
  - (Ajax-запити до сервера можна додати при наявності backend API)

## Підсумок

В результаті 5 лабораторних робіт створено повноцінний сайт пекарні Gustese з:

- Адаптивною версткою та мобільним меню
- Системою реєстрації та входу на всіх сторінках
- Інтерактивною таблицею контактів (пошук, сортування, пагінація)
- CRUD-операціями для авторизованих користувачів
- Збереженням даних у localStorage (записи не зникають після виходу)
- Стилізованими елементами DataTables у відповідності до дизайну сайту
